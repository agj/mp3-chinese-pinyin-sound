
mp3-chinese-pinyin-sound
========================

Mandarin Chinese pinyin sounds in MP3 format


## File naming

Sounds for tones 1 through 4 are included, named `ling1.mp3` for 'līng', `luu4.mp3` for 'lǜ', etc.
